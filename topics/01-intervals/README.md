# Intervalos

**Status:** não iniciado

## Objetivo

Reconhecer intervalos de ouvido, sempre enquadrados por um centro tonal (nunca nota A→B isolada) — a base de toda percepção auditiva de harmonia de jazz que vem depois.

## Pré-requisitos

Nenhum — este é o tema-base do projeto.

## Praticar

[Praticar →](widgets/interval-drill.html)

Lição de apoio: [Graus e centro tonal](lessons/01-intro-graus-e-centro-tonal.md)

## Métricas

Placar da sessão no widget: `acertos/tentativas` (ex. `7/10`) — reinicia a cada abertura da página, sem histórico de % total persistido. Sequência de acertos seguidos também é exibida.

**Confusões frequentes:** o widget registra em `localStorage` (chave `intervals-confusions`) os pares intervalo-certo → intervalo-respondido sempre que você erra, e mostra os mais recorrentes na tela (ex. "2ª menor → você respondeu 3ª menor"). Confusões recorrentes são boas candidatas a virar card manual em `review/box-1-ainda-nao-sei/` (criação do card continua manual — ver `review/README.md`).
